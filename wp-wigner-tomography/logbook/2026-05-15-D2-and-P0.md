### 1.1 D2 reach ladder

For each $N \in \{20, 40, 60, 80\}$ with $|\beta_0| = 0.05$, the
accessible-disk radius is $B_N = N|\beta_0| \in \{1, 2, 3, 4\}$. On
the $81 \times 81$ Cartesian β grid spanning $[-4, 4]^2$ with
$\Delta\beta = 0.10$:

| $N$ | $B_N$ | predicted nodes inside disk | rough estimate |
|---:|---:|---:|---|
| 20 | 1.0 | $\sim \pi \cdot 1^2 / \Delta\beta^2 \approx 314$ | $\pi B_N^2/\Delta\beta^2$ |
| 40 | 2.0 | $\sim 1257$ | |
| 60 | 3.0 | $\sim 2827$ | |
| 80 | 4.0 | $\sim 5027$ | matches the WP-W §2 stated value 5025/6561 |

For the coherent state $|\alpha=1\rangle$ the characteristic function
$\chi(\beta) = e^{-|\beta|^2/2} \cdot e^{2i\,\mathrm{Im}(\alpha^*\beta)}$
gives $|C(\beta)| = e^{-|\beta|^2}$ — an isotropic Gaussian falloff
centred at the origin, independent of $\alpha$ and $N$. The N-dependence
shows only through the accessible-disk envelope.

$\mathrm{Re}\,\chi(\beta)$ for $\alpha=1$ has the form
$e^{-|\beta|^2/2}\cos(2\beta_y)$ — oscillations along the $\mathrm{Im}\,\beta$
direction with period $\pi/1 \approx 3.14$ (one full sign-change cycle
between $\beta_y \approx -1.5$ and $+1.5$), the *Wigner-tomography
phase information* encoded in the protocol.

### 1.2 P0 analytic-grid self-consistency

Per the §4a pass criteria:

- **Vacuum**: $W(\alpha = 0) = 2/\pi \approx 0.6366$ to within FFT
  precision; no sign flip; peak at the origin within $\Delta\alpha$.
- **Coherent $|\alpha=1\rangle$**: peak at the grid point closest to
  $\alpha=1$ (which is $\alpha \approx 1.1636$ for $\Delta\alpha = \pi/(N\Delta\beta) = 0.388$);
  peak value $= (2/\pi)\exp(-2|1.1636 - 1|^2) = 0.6035$; no sign flip.

Pass criterion: $|\alpha_\text{peak} - \alpha_\text{true}| < \Delta\alpha \approx 0.388$.

### 1.3 FFT convention sanity check

Implementation: ifftshift → ifft(axis=1, positive sign for $\beta_x \to \alpha_y$) → fft(axis=0, negative sign for $\beta_y \to \alpha_x$) → fftshift → transpose for imshow consistency. Expected
$\max |\mathrm{Im}\,W| < 10^{-12}$ for analytically pure χ.

### 3.2 P0 self-consistency

| state | metric | predicted | measured | flag |
|---|---|---:|---:|:-:|
| vacuum | $W(\alpha=0)$ | $2/\pi = 0.636620$ | $0.636555$ | ✓ ($\Delta = -6.5\times 10^{-5}$) |
| vacuum | peak offset | $\leq \Delta\alpha = 0.388$ | $0.0000$ | ✓ |
| vacuum | max $\|\mathrm{Im}\,W\|$ | $\lesssim 10^{-12}$ | $1.5 \times 10^{-16}$ | ✓ |
| coherent | peak position | $(1, 0)$ → closest grid pt $(1.1636, 0)$ | $(1.1636, 0)$ | ✓ exactly on the closest grid point |
| coherent | peak offset from $\alpha_\text{true}$ | $\leq \Delta\alpha = 0.388$ | $0.1636$ | ✓ |
| coherent | $W$ at peak | $(2/\pi)e^{-2(0.1636)^2} = 0.60346$ | $0.60342$ | ✓ ($\Delta = -3.7 \times 10^{-5}$) |
| coherent | sign flip | none | none | ✓ |
| coherent | max $\|\mathrm{Im}\,W\|$ | $\lesssim 10^{-12}$ | $3.6 \times 10^{-16}$ | ✓ |

**P0 gate: PASS** (overall).

The $-6.5\times 10^{-5}$ residual on the vacuum centre value is
finite-grid windowing (the Gaussian $\chi_\text{vac}$ has support
slightly outside $|\beta| = B = 4$; the grid truncation costs ~$10^{-7}$
in the integral, of which a fraction shows up as a small numerical
underestimate at the centre). The discrepancy is far below any
practical reconstruction-quality threshold.

The coherent-state peak landing **exactly** on the predicted closest
grid point — and at the right value to one part in $10^4$ — confirms
the FFT convention (sign of $\beta_x \to \alpha_y$ vs. $\beta_y \to \alpha_x$)
and the chosen $\Delta\beta\,\Delta\alpha = \pi/N$ relation in §2.

### 3.3 FFT convention

`max |Im W|` for both vacuum ($1.5 \times 10^{-16}$) and coherent
($3.6 \times 10^{-16}$) are at the machine-epsilon floor for `complex128`,
confirming the convention is implemented correctly. No phase-twist
residual was observed.

**One mid-session correction.** The first plot run revealed that my
initial `wigner_from_chi` output had `rows = α_x, cols = α_y` — opposite
to the input χ convention `rows = Im β, cols = Re β`. The peak
appeared visually rotated by 90° relative to the legend annotation.
Fix: transpose the output in `wigner_from_chi` so the output convention
mirrors the input. After the fix, the coherent-state peak landed where
expected on the displayed axes. **Lesson logged**: array-axis
↔ physical-axis conventions need explicit type-tagging in the docstring.
