quasiprobability distributions, unifying the Glauber–Sudarshan $P$
($s=+1$), Wigner $W$ ($s=0$), and Husimi $Q$ ($s=-1$) representations
through a single parametrised Fourier-transform pair with the
displacement operator $\hat D(\lambda) = e^{\lambda\hat a^\dagger - \lambda^*\hat a}$.

WP-W's measurement chain — the chain that says (i) the symmetric
characteristic function $\chi(\beta) = \langle\hat D(\beta)\rangle$ is
a complete state representation, (ii) the Wigner function is the 2D
Fourier transform of $\chi$, and (iii) both apply to *any* density
operator including non-Gaussian states. WP-W §Analytical bullets 2,
3, and 4 cite [CG69] for these three points.

## 1. Core formalism (standard CG69 framework)

### 1.1 The displacement operator

$$\hat D(\lambda) = \exp\!\bigl(\lambda\hat a^\dagger - \lambda^* \hat a\bigr)$$

This unitary shifts the coherent-state label: $\hat D(\lambda)|\alpha\rangle = e^{(\lambda\alpha^* - \lambda^*\alpha)/2}|\alpha + \lambda\rangle$.

### 1.2 $s$-ordered characteristic function

For a density operator $\hat\rho$ and complex ordering parameter $s$,

$$\chi(\lambda, s) = \mathrm{Tr}\!\bigl[\hat\rho\,\hat D(\lambda)\bigr]\,e^{s|\lambda|^2/2}.$$

Special cases:
- $s = +1$: normal-ordered, Glauber–Sudarshan $P$ characteristic.
- $s = 0$: **symmetric (Weyl)** characteristic — *this is the function WP-W measures*.
- $s = -1$: anti-normal-ordered, Husimi $Q$ characteristic.

The symmetric ($s=0$) characteristic function is the expectation of the displacement operator itself:

$$\boxed{\;\chi(\lambda, 0) = \langle\hat D(\lambda)\rangle = \mathrm{Tr}\!\bigl[\hat\rho\,\hat D(\lambda)\bigr]\;}$$

**This is exactly the quantity FH20 measures and WP-W proposes to measure**, up to the bichromatic-vs-stroboscopic distinction.

### 1.3 $s$-ordered quasiprobability distribution

The corresponding quasiprobability is the 2D Fourier transform of the $s$-characteristic function:

$$
W(\alpha, s) = \frac{1}{\pi^2}\!\int \chi(\lambda, s)\,e^{\alpha\lambda^* - \alpha^*\lambda}\,d^2\lambda.
$$

Special cases:
- $s = +1$: $W(\alpha, +1) = P(\alpha)$ — Glauber–Sudarshan $P$ representation.
- $s = 0$: $W(\alpha, 0) = W(\alpha)$ — **Wigner function**.
- $s = -1$: $W(\alpha, -1) = Q(\alpha) = \langle\alpha|\hat\rho|\alpha\rangle/\pi$ — Husimi $Q$.

**WP-W §Analytical bullet 3** uses exactly the $s=0$ specialisation of this
inversion: $W(\alpha) = \pi^{-2}\!\int \chi(\beta)\,e^{\alpha\beta^* - \alpha^*\beta}\,d^2\beta$.

### 1.4 State-generality

CG69 establishes that the $s$-ordered characteristic function $\chi(\lambda, s)$
is a *complete* state representation: any density operator $\hat\rho$
is uniquely recoverable from its $\chi(\lambda, s)$ for any fixed $s$
(via the Fourier inversion of §1.3 to obtain the $s$-ordered
quasiprobability, then taking moments). This holds *whether or not*
$\hat\rho$ is a pure state, Gaussian, or even a well-behaved positive
function in phase space — the formalism is state-general by construction.

**WP-W §Analytical bullet 4** cites [CG69] for this state-generality claim.

## 2. What [CG69] does NOT do

CG69 is *pure formalism*. It does not:

- Discuss measurement protocols (no Ramsey, no displaced parity, no SDF).
- Address experimental implementation in any physical system.
- Treat the trapped-ion or cavity-QED specialisations that come decades later via [LD97], [LMKMIW96], [Bertet02], [Hof09], [FH20].
- Discuss the *high-η* Lamb–Dicke perturbative chain or the stroboscopic-comb structure central to WP-W.

CG69's role is to *legitimise* the χ-and-W formalism as state-general
mathematics — providing the framework against which the experimental
protocols of LD97/FH20/WP-W can be benchmarked.
