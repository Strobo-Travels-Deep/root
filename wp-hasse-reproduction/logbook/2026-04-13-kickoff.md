## 1. Convention mapping decisions

### Analysis-phase ϕ axis (Hasse Fig 6)

The engine returns post-train (⟨σ_x⟩, ⟨σ_y⟩, ⟨σ_z⟩) in a fixed frame.
Hasse's analysis phase ϕ is reconstructed by basis rotation, not by an
extra simulation axis:

```
σ_z(ϕ) = ⟨σ_x⟩ cos(ϕ) + ⟨σ_y⟩ sin(ϕ)
ϕ₀     = arg(C)         where C = ⟨σ_x⟩ + i ⟨σ_y⟩
|C|    = √(σ_x² + σ_y²)
```

This is exact under the standard Ramsey readout convention. **No engine
patch.**

### LF mode tilt → effective η (Hasse Fig 8)

Hasse Fig 8 caption: "calibrations for 0° (blue), −5° (green), and +5°
(orange)" of the LF mode angle relative to the z axis. The projection
onto the AC wave-vector scales as

```
η(angle) = η_LF × cos(angle)
```

so the three values used in `run_fig8.py` are
{0.397·cos(−5°), 0.397, 0.397·cos(+5°)} ≈ {0.3955, 0.397, 0.3955}.
The asymmetry between ±5° in Hasse Fig 8 must therefore arise from the
*sign* of an additional projection (e.g. orientation-dependent sign of
the linear coupling), not from |cos|. Resolving this exactly is deferred
to the results entry; for now we run the symmetric mapping and document
the deviation.

### Back-action δ⟨n⟩

Engine returns `nbar` (final ⟨n⟩) per detuning point. Initial ⟨n⟩ is
computed analytically from the prepared coherent state |α e^{iϑ₀}⟩:
⟨n⟩_ini = |α|². Then δ⟨n⟩ = nbar − |α|².

## 2. Run plan

Sequence (single pass, no preflight gating beyond engine import):

1. Write `run_fig6.py` — fix |α| = 3, η = 0.397, n_thermal = 0; sweep
   `alpha_phase_deg` ∈ [0, 360°] at high resolution (~64 points); for
   each prepared state, run `run_single` with `det_min = det_max = 0`,
   `npts = 1` to read out (σ_x, σ_y, σ_z, nbar). Reconstruct Hasse Fig 6
   panels by ϕ-basis rotation.
2. Write `run_fig8.py` — three η values; sweep |α| ∈ [0, 8] (Hasse y-axis
   range) at ~17 points; n_thermal = 0.15. For each, extract ϕ₀ = arg C
   and contrast |C|.
3. Run both scripts, write HDF5.
4. Plot Fig 6a (heatmap + 3 cuts), Fig 6b (heatmap + 3 cuts), Fig 8a/b.
5. Write results entry with anchor-number comparison.
