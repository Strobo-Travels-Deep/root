One-Sentence Claim

Hasse et al. phase-lock four oscillators (MW reference, optical running-wave phase, spin, motion) and use a stroboscopic Raman pulse train on a short-period travelling polarisation gradient (λeff ≈ 140 nm, ηLF ≈ 0.40) to map an ion's motional phase-space dynamics with super-resolution, where position appears as a spin-fringe phase shift and the p-quadrature proxy appears through the characteristic Doppler-detuning spectrum of the analysis pulse, encoding the ion's velocity distribution.

Glossary

Momentum observable
A velocity / p-quadrature proxy inferred from the Doppler-detuning response of the stroboscopic analysis pulse.
Super-resolution
Position sensitivity at λeff ≈ 140 nm ≪ optical wavelength. Does not imply sub-x0 or QND.
Sideband scaling ~ηs
Heuristic. Actual weights depend on n via Laguerre polynomials.
Detuning spectrum
P↑(δ) as function of analysis-pulse detuning. Encodes velocity distribution via Doppler shift keff·v.

RESOLVED WP-E §Analytical — §2.1 normalises |C| against the reference state at (α=0, δ0=0, φα=0) rather than against an idealised π/2, so the 1.53 vs π/2 gap does not enter residuals.

Linear LD picture requires η√(⟨n⟩+1) ≲ 1; at η = 0.397 this fails for |α| ≥ 3 (1.26 ✗) and |α| = 5 (2.03 ✗). Evidence Summary

Experimentally established

Four-oscillator phase lock. AC Raman: λeff ≈ 140 nm, Ω/(2π) ≈ 0.3 MHz. ηLF ≈ 0.40, ηMF ≈ 0.23, ηHF ≈ 0.18. Timing: tπ/2,S ≈ 23.1 μs.

Analytically grounded

Coherent-state baseline: exp(−η²/2)·exp(i2η Re α). Sideband amplitudes from |0⟩: carrier 0.923, 1st 0.369, 2nd 0.074, 3rd 0.010.

Numerical calibration only

Full decoder for arbitrary states. Backaction beyond ⟨n⟩. Non-Gaussian behaviour. See Numerics · Simulate · Code for interactive data.
