Method

Full documentation: Code page. Initial motional state prepared as S(r,φs)D(α eiθα)|n⟩, with thermal averaging over Fock states when ⟨n⟩th > 0. Squeeze operator built exactly via matrix exponentiation of (r/2)(e−2iφa² − e2iφa†²). Sidebands at δ₀ = ±s·ωm emerge naturally from the exact exp(iη(a+a†)) coupling.
