Method

Full spin–motion Schrödinger equation in truncated Fock basis. The coupling operator C = exp(iη(a+a†)) is computed exactly — no Lamb–Dicke or sideband truncation. This naturally produces the full sideband spectrum: for the ground state (α=0), sharp features appear at δ₀ = ±s·ωm with weights governed by the Laguerre-weighted sideband matrix elements. For displaced states (α>0), the Doppler mechanism broadens the spectrum into the characteristic velocity-distribution envelope described in the Tutorial.

Pulse timing: Np pulses separated by Tm = 2π/ωm. Per-pulse duration δt = π/(2NpΩeff) ensures total π/2 rotation. Free evolution exp(−iωma†a·Tm) = I (exact in Fock basis). See Code for full documentation.
