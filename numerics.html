Data Format

Each JSON file contains a manifest (parameters, provenance SHA-256 hash, convergence diagnostics) and a data object with arrays: detuning, sigma_x, sigma_y, sigma_z, coherence, entropy. Files from runs with projection noise additionally include noisy_sigma_* and err_sigma_*.

Generate your own results on the Simulate page, or use the downloadable Python/QuTiP script for batch runs.
