Simulation: each pulse applies exp(−iH·δt) where δt = (π/2)/(Npulses·Ωeff). Free evolution between pulses = identity (exact stroboscopic sync).

Data Format

HDF5 files contain detuning, sigma_x/y/z, entropy, coherence arrays plus root-group attributes. JSON files from Simulate contain a manifest (parameters, provenance hash, convergence) and the same data arrays. See scripts/export_hdf5.py for the HDF5 export pipeline.
